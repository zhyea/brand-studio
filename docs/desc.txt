[en] English
--------------------------------------------------------------------------------
BrandStudio (Showcase Capture Studio) is a Chrome extension that captures your current browser tab and turns it into a polished promotional or showcase image.

How to use
1. Open the page you want in the tab.
2. Click the extension icon on the toolbar. Chrome captures the visible area of that tab and opens the editor (or focuses it if it is already open).
3. Pick a layout, a background (gradients, solid colors, built‑in textures, or your own image), title and subtitle text (font, size, color, transparency, effects), and optional arrows.
4. Optionally show an alignment grid, wrap the screenshot in a decorative browser frame, and choose the frame color.
5. Preview the result, then export a PNG. Presets include common store promo sizes (e.g. thumbnail, standard, HD); you can also set custom width and height.

Features (summary)
• Several composition layouts (split text left/right, text above or below the shot, narrow split, full bleed, compact stack, etc.).
• Rich backgrounds and optional grid overlay while editing.
• Text styling that matches the exported canvas as closely as possible in the editor.
• Arrows with multiple head shapes and line styles.
• Light and dark UI themes; many choices (grid, embed browser, background, text style, arrow defaults, etc.) are saved in local storage and kept when you capture a new tab.
• Session storage briefly holds the latest capture so the editor can open right after you click the icon.

Privacy & data
The extension captures only what you trigger via the toolbar action. The image is passed to the editor tab on your device. Preferences are stored locally in Chrome (local/session storage); nothing is sent to a remote server by this extension.

BrandStudio（Showcase Capture Studio）是一款 Chrome 扩展，用于截取当前浏览器标签页的可见区域，并制作成适合应用商店、宣传物料或作品展示的配图。

使用方法
1. 在要展示的标签页中打开目标网页。
2. 点击工具栏上的扩展图标。Chrome 会截取该标签页的可见区域，并打开编辑器标签页（若已打开则会切到该页）。
3. 选择版式、背景（渐变、实色、内置纹理或自定义图片）、主标题与副标题（字体、字号、颜色、透明度、效果），并可添加箭头标注。
4. 可选：显示对齐网格、为截图套上装饰性「浏览器窗口」外框并设置外框颜色。
5. 预览合成效果后导出 PNG。内置多种商店推广常用尺寸（如缩略图、标准、高清），也支持自定义宽高。

功能概要
• 多种版式（左右分栏、标题在上下方、窄分栏、通栏铺满、紧凑顶栏等）。
• 丰富背景与编辑时可选网格。
• 文案样式在编辑区尽量接近最终画布导出效果。
• 多种箭头头部与线条样式。
• 亮色 / 暗色主题；网格、嵌入浏览器、背景、文案样式、箭头等多数选项会保存在本机，更换新截图时仍会保留。
• 新截取的画面通过会话存储短暂传递以便立刻进入编辑。

隐私与数据
仅在您点击工具栏按钮时截取当前标签页可见内容；数据在您本机的浏览器中处理。偏好保存在 Chrome 本地存储中；本扩展不向自有服务器上传截图内容。

• विभिन्न तीर-सिर और रेखा शैलियाँ।
• हल्का/गहरा थीम; कई सेटिंग्स स्थानीय रूप से सहेजी जाती हैं।
• नवीनतम कैप्चर सत्र संग्रह में संक्षिप्त रखा जाता है।

गोपनीयता
कैप्चर केवल जब आप बटन से चालू करें। डेटा आपके डिवाइस पर संसाधित; प्राथमिकताएँ Chrome स्थानीय संग्रह में। यह एक्सटेंशन स्क्रीनशॉट अपने सर्वर पर नहीं भेजता।
